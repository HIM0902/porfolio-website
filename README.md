## 💡 Overview
This portfolio was engineered to be fast, responsive, and visually engaging. It utilizes a dark-mode, glassmorphism aesthetic with tailored 2D physics and staggered animations to guide the user experience.

**Key Featured Projects:**
* **Tech Price Forecaster:** An end-to-end ML pipeline using XGBoost and Azure Blob Storage to predict semiconductor costs.
* **YouTube Intelligence Platform:** Scalable data ingestion and ETL pipelines for trend forecasting.
* **Dead Silence (Unity AI):** A 3D survival game emphasizing state-machine-driven pathfinding and OOP design patterns.

## 🛠️ Technical Stack
**Frontend Architecture:**
* **React 18** - Component-based UI rendering
* **Vite** - Next-generation frontend tooling and bundling
* **Framer Motion** - Production-ready declarative animations (Staggered arrays, 2D transforms)
* **Custom CSS** - Scoped variable-driven styling (Glassmorphism, CSS Grid/Flexbox)

**Deployment:**
* **GitHub Pages** - Automated CI/CD deployment
